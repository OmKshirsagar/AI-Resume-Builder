# Tech Stack: Gemini Resume Builder

## Frameworks
- **Next.js 15.5+**: React framework with App Router and Turbopack.
- **React 19**: Modern UI library with stable Server Actions.

## AI & Agents
- **Mastra**: Multi-agent framework for reliable document fabrication.
- **Vercel AI SDK**: For streaming and provider-agnostic model integration.
- **Google Generative AI**: Access to Gemini and Gemma models.

## Styling & UI
- **Tailwind CSS 4.0**: High-performance CSS utilities.
- **React Resizable Panels**: For the split-pane editor/preview interface.
- **Lucide React**: Icon system.

## State & Forms
- **Zustand 5**: Client-side state management with middleware persistence.
- **React Hook Form**: Multi-section form orchestration.
- **Zod**: Runtime schema validation.

## Document Generation
- **@react-pdf/renderer**: Client-side PDF engine.
- **file-saver**: Cross-browser download management.

## Quality & Tooling
- **Biome**: Linter and formatter (replacement for ESLint/Prettier).
- **TypeScript 5.8+**: Strict type safety.
- **Vitest**: Test runner.
